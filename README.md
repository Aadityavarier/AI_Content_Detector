# Veridian — AI Content Detector

> Detect AI-generated text, audio, and video instantly.
> Free, open-source, no signup required.

**Live Site** → [veridion-omega.vercel.app](https://veridion-omega.vercel.app)
**API Docs** → [veridian-backend-cglg.onrender.com/api/docs](https://veridian-backend-cglg.onrender.com/api/docs)

---

## Overview

Veridian is a full-stack AI content detection platform that analyzes text, audio, and video to determine whether the content was generated by AI. Built entirely with free and open-source tools, it runs a statistical linguistic analyzer for text, wav2vec2 + MFCC feature extraction for audio, and EfficientNet-B0 frame-level analysis for video.

---

## Features

- **Text Detection** — Analyzes 13+ linguistic patterns including sentence uniformity, vocabulary richness, AI phrase density, contraction usage, and first-person pronoun frequency
- **Audio Detection** — Extracts MFCC features and spectral characteristics to identify synthetic or cloned voices
- **Video Detection** — Samples frames with OpenCV and runs EfficientNet-B0 to detect deepfakes
- **Instant Results** — Probability score (0–100%), confidence rating, and model metadata returned per request
- **Mobile Responsive** — Full hamburger menu and responsive layout for all screen sizes
- **Rate Limited** — 20 req/min for text, 10 for audio, 5 for video
- **File Validation** — Strict format and size (50MB) checks on all uploads

## Local Setup

### Prerequisites
- Python 3.11 (Anaconda recommended)
- Node.js 20+
- Git

### Backend

```bash
cd backend
conda create -n veridian python=3.11 -y
conda activate veridian
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

API → `http://localhost:8000`
Swagger UI → `http://localhost:8000/api/docs`

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Site → `http://localhost:3000`

Create `frontend/.env.local`:
```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

## API Reference

### POST `/detect/text`
```bash
curl -X POST https://veridian-backend-cglg.onrender.com/detect/text \
  -F "text=Your text content here minimum fifty characters long..."
```

Response:
```json
{
  "id": "uuid",
  "type": "text",
  "ai_probability": 0.84,
  "human_probability": 0.16,
  "confidence": "high",
  "model_used": "statistical-linguistic-v2",
  "label": "AI Generated",
  "analyzed_at": "2025-03-15T10:30:00"
}
```

### POST `/detect/audio`
```bash
curl -X POST https://veridian-backend-cglg.onrender.com/detect/audio \
  -F "file=@speech.mp3"
```

Supported formats: `mp3, wav, ogg, flac, m4a` — Max 50MB

### POST `/detect/video`
```bash
curl -X POST https://veridian-backend-cglg.onrender.com/detect/video \
  -F "file=@video.mp4"
```

Supported formats: `mp4, mov, avi, mkv, webm` — Max 50MB

## Deployment

### Frontend → Vercel

1. Push repo to GitHub
2. Go to vercel.com → New Project → Import repo
3. Set Root Directory to `frontend`
4. Set Framework to `Next.js`
5. Add environment variable: `NEXT_PUBLIC_API_URL` = your Render URL
6. Deploy

### Backend → Render

1. Go to render.com → New Web Service → Connect repo
2. Set Root Directory to `backend`
3. Set Build Command: `pip install -r requirements.txt`
4. Set Start Command: `uvicorn main:app --host 0.0.0.0 --port $PORT`
5. Add environment variable: `FRONTEND_URL` = your Vercel URL
6. Deploy

### Database → Supabase

1. Create free project at supabase.com
2. Copy PostgreSQL connection string
3. Change `postgresql://` to `postgresql+asyncpg://`
4. Add as `DATABASE_URL` environment variable on Render

## Detection Accuracy

| Modality | Method | Notes |
|---|---|---|
| Text | Statistical linguistic v2 | 13 features — phrase density, TTR, uniformity |
| Audio | wav2vec2 + MFCC heuristic | Spectral + cepstral feature analysis |
| Video | EfficientNet-B0 | Frame sampling via OpenCV, aggregated score |

> For higher text accuracy, replace `text_detector.py` with the RoBERTa version (requires 1GB+ RAM on server).
